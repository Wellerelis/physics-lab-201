## Что делает калькулятор

Вы вводите результаты измерений (ВАХ и ХЭДС), калькулятор автоматически строит:

- **График ВАХ** — I(U‖) с линией МНК и линией метода парных точек
- **График ХЭДС** — U⊥(B) с линией МНК и линией метода парных точек
- **Таблицу 3** — вычисление удельной проводимости σ методом парных точек (по форме из методички)
- **Таблицу 4** — вычисление константы Холла R методом парных точек (по форме из методички)
- **Погрешности** — полная цепочка δR, δσ, δn, δμ с разбивкой по источникам
- **Итоговые результаты** — R ± ΔR, σ ± Δσ, n ± Δn, μ ± Δμ
- **Протокол** — готовый текст для вставки в отчёт

## Физика

### Константа Холла

Поперечное напряжение (ХЭДС) линейно зависит от магнитной индукции:

```
U⊥ = R · I · B / d
```

Из графика U⊥(B) находится угловой коэффициент `a = dU⊥/dB`, откуда:

```
R = a · d / I
```

### Удельная проводимость

Из графика ВАХ I(U‖) находится угловой коэффициент `a = dI/dU‖`, откуда:

```
σ = a · l / S
```

### Концентрация и подвижность носителей

```
n = 1 / (e · |R|)

μ = σ · |R|
```

### Погрешности

```
δR = √(δa² + δd² + δI²)          — a из ХЭДС, d — толщина, I — ток
δσ = √(δa² + δl² + δS²)          — a из ВАХ, l — длина, S — сечение
δn = δR
δμ = √(δσ² + δR²)
```

## Как пользоваться

### 1. Параметры

Проверьте значения геометрии образца и коэффициента электромагнита K. Все производные величины (S = d·h, l/S, d/I) вычисляются автоматически и показаны в сводке.

### 2. ВАХ (Табл. 1)

Введите пары (U‖, I) для каждого опыта — от 0.2 до 3.0 мА с шагом 0.2 мА (16 строк). Вольтметр V1, диапазон 20 В; миллиамперметр mA, диапазон 20 мА.

### 3. ХЭДС (Табл. 2)

Введите ток магнита I_M (А) и поперечное напряжение U⊥ (мВ) для каждого опыта — от 0.02 до 0.22 А с шагом 0.02 А (11 строк). Вольтметр V2, диапазон 200 мВ. Индукция B = K·I_M рассчитывается автоматически.

### 4. Табл. 3 & 4

Автоматически строятся таблицы вычислений σ и R методом парных точек — точно по форме из методички. Для каждой таблицы можно выбрать режим разбивки на пары:

| Кнопка | Описание |
|--------|----------|
| 1…n | Все точки, пары (1, ⌊n/2⌋+1), …, (⌊n/2⌋, n) |
| 2…n | Пропустить первую точку |
| 3…n | Пропустить первые две точки |

При нечётном числе точек средняя автоматически исключается из разбивки.

### 5. Обработка

Два графика с кнопками выбора шага осей (**авто**, **×1**, **×2**, **×5**), таблицы метода парных точек с погрешностями, итоговые значения всех четырёх физических величин.

### 6. Протокол

Кнопка «Скопировать» копирует готовый текст протокола в буфер обмена.

## Формулы

### МНК

```
a = (n·Σxᵢyᵢ − Σxᵢ·Σyᵢ) / (n·Σxᵢ² − (Σxᵢ)²)
σ_a = s / √(Σ(xᵢ − x̄)²)
```

### Метод парных точек (раздел 8 методички)

```
aᵢ = (y_{i+k} − yᵢ) / (x_{i+k} − xᵢ)     i = 1…k

σ⟨a⟩ = √( Σ(aᵢ − ⟨a⟩)² / (k·(k−1)) )

Δa = t(P=0.90, f=k−1) · σ⟨a⟩
```

Коэффициенты Стьюдента взяты из таблицы при P = 0.90.

### Погрешность S

```
δS = √( (δd/d)² + (δh/h)² )
```

## Известные ограничения

- Параметры стенда (d, l, h, K) захардкожены для установки ДХК-0.5 кафедры физики СПбПУ — при работе на другом стенде измените значения во вкладке «Параметры»

- Коэффициент Стьюдента рассчитан для P = 0.90; для других доверительных вероятностей потребуется изменить таблицу в коде
