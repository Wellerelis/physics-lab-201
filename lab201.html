Уравнение (1) — ХЭДС
U⊥ = R · (I·B) / d
R — константа Холла [м³/Кл], d — толщина образца, I — ток, B — индукция поля
Уравнение (2) — Закон Ома
I = U‖ / r = σ · (S/l) · U‖
σ — удельная проводимость, S = d·h — площадь сечения, l — длина
Уравнения (5)–(6) — Проводимость
j = σ · E‖ = σ · (U‖/l)    σ = 1/ρ = (I/U‖) · (l/S)
Из ВАХ: σ = (dI/dU‖) · (l/S) — наклон × геометрия
Уравнение (12) — Константа Холла из эксперимента
R = U⊥ · d / (I · B) = (dU⊥/dB) · d / I
Из графика U⊥(B): угловой коэффициент a = dU⊥/dB → R = a·d/I
Уравнение (13) — Концентрация носителей
n = 1 / (e · R)
e = 1.6×10⁻¹⁹ Кл
Уравнение (14) — Подвижность носителей
μ = σ · R
μ — подвижность [м²/(В·с)], σ — из ВАХ, R — константа Холла

δn = δR    δμ = √[(δσ)² + (δR)²]
